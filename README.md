# Simulador de carrinhos de compras
<br>

##  ✍️ Sobre
<p>Criação de um simulador de carrinhos de compras, onde os valores são armazenados e vão aumentando quanto mais produtos selecionados. Manipulação do DOM via JavaScript.</p>
<p>Parte da formação da Alura. Layout e estilos fornecidos pela plataforma.</p>
<br>

## 🎥 Funcionalidades
* **Cálculo dos produtos:**  Armazena a quantia dos valores do produtos e soma conforme mais itens forem sendo adicionados, retornando o valor total
* **Limpeza do carrinho:** Uma função que limpa o carrinho inteiro, resetando todos os itens e preços
